# Portfólio de Música - Marlon Lotici

## Sobre o Projeto
Este é o meu portfólio pessoal como músico e produtor, criado para exibir meu trabalho, agenda de shows e informações de contato. O site é uma página única, responsiva e otimizada para ser visualizada em qualquer dispositivo.

## Tecnologias Utilizadas
- **HTML5:** Para a estrutura e conteúdo da página.
- **CSS3:** Para o design e a estilização, garantindo uma aparência moderna.
- **Formspree:** Para o envio seguro de mensagens a partir do formulário de contato, sem a necessidade de um servidor próprio.
- **Netlify:** Para hospedagem e publicação contínua do site.

## Como o Projeto Funciona
O site é um portfólio estático, o que significa que ele não utiliza um banco de dados. Para enviar mensagens do formulário de contato, ele se conecta ao serviço Formspree, que encaminha as mensagens para o meu e-mail.
